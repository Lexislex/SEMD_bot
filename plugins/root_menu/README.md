# Root Menu Plugin

## Описание

Плагин главного меню бота. Обеспечивает центральную маршрутизацию и доступ ко всем остальным плагинам.

## Команды

- `/start` — открыть главное меню

## Кнопки

- Автоматически показывает кнопки всех доступных плагинов
- « Назад в меню — вернуться в главное меню

## Доступ

- Доступен для всех пользователей (`access_level = "all"`)
